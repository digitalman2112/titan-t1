## Setup

1. Remove the <span class="ref-accessory">exercise handles</span> from the smith carriage.
2. Release the <span class="ref-accessory">pec deck arms</span> from the storage <span class="ref-hook">hooks</span> so that they rest hanging downward in the exercise position as shown in the picture.
3. Hook the <span class="ref-cable">pec deck cable</span> <span class="ref-cable">C11</span> onto the locking pin located at the center of the smith carriage crossbar.
4. Incline the backrest to approximately 80° and lock the pull pin located underneath the backrest.
5. Adjust the seat to the desired incline position and lock into place using the pull pins underneath. Slide the bench to position **5** (adjust to preference) and lock into place.
6. Insert <span class="ref-accessory">attachment P-1</span> seat into hole **27** located underneath the seat.

## Pec Deck

1. Assume a comfortable seated position with the back resting against the backrest. Plant the feet firmly on the ground or use the provided footrest.
2. Firmly grasp the handles in each hand.
3. In a slow, controlled manner, squeeze the grips together in front of the chest with the arms slightly bent. Pause briefly and return to the starting position.
4. If the weight of the smith carriage without added weight is too heavy, insert <span class="ref-cable">counter-balance cable</span> <span class="ref-cable">C2</span> into hole **18** for counter-balance and then add the desired counter weight.
5. Upon completion of the exercise, lower weight gently to the original starting position.
6. Instructions can also be found on the video CD that was included with your machine.

## Exercise Precautions

- After the bench is properly positioned, ensure that it is locked into place.

## After Completing

1. If the counter-balance was used, remove any weight from the weight post and then remove <span class="ref-cable">C2</span> cable from hole **18** and return to the storage hole.
2. Remove the <span class="ref-accessory">seat attachment P-1</span>.
3. Remove <span class="ref-cable">cable</span> <span class="ref-cable">C11</span> from the smith carriage crossbar and attach to the storage for C-11 pin.
4. Take the <span class="ref-accessory">pec deck arms</span> back onto the <span class="ref-hook">hooks</span>.
