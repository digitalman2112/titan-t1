## Setup

1. Using an attachment clip, attach <span class="ref-cable">cable</span> <span class="ref-cable">C8</span> to the attachment bar located on the inside of the counter-balance.
2. Attach <span class="ref-cable">extension cable</span> <span class="ref-cable">C5</span> to the other end of <span class="ref-cable">cable</span> <span class="ref-cable">C8</span>.
3. Attach <span class="ref-accessory">handle</span> <span class="ref-accessory">S8</span> to the <span class="ref-cable">extension cable</span> <span class="ref-cable">C5</span>.
4. Add the desired weight to the counter balance weight post.

## Tricep Push Downs

1. Grab the <span class="ref-accessory">handle</span> <span class="ref-accessory">S8</span> firmly with both hands.
2. Push downward with the elbows in close to the waist, pausing at the near extended position, and then return to the starting position.
3. Instructions can also be found on the video CD that was included with your machine.

## Precautions

- Be sure that all cables are properly and securely attached before use.
