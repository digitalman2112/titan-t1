## Setup

1. Hook <span class="ref-cable">lat pulley cable</span> <span class="ref-cable">C3</span> onto the locking pin at the center of the smith carriage crossbar. Remove the <span class="ref-accessory">exercise handles</span> from the smith carriage if necessary, and any other handles or attachments.
2. Using a clip, attach <span class="ref-accessory">tricep push-down bar</span> <span class="ref-accessory">S8</span> to <span class="ref-cable">C3</span> hanging overhead at the forefront of the machine. If the bar is too high to grab, use <span class="ref-cable">attachment cable</span> <span class="ref-cable">C5</span> or an additional carabiner clip to lengthen <span class="ref-cable">C3</span>.
3. Add the desired weight and be sure the weights are evenly distributed on each side of the machine.
4. Raise the thigh roller pads to the upright position. Be sure the pull pin is fully engaged and locked in position.
5. Adjust the seat to the desired position and lock into place.
6. If the weight of the smith carriage without added weight is too heavy, insert <span class="ref-cable">counter-balance cable</span> <span class="ref-cable">C2</span> into hole **18** and add the desired counter weight.

## Triceps Push Downs

1. Stand facing the machine and firmly grasp the <span class="ref-accessory">handle</span> <span class="ref-accessory">S8</span>.
2. In a slow and controlled manner, push the bar downward with elbows close to the waist.
3. Pause briefly in the fully contracted position and return slowly to the starting position.

## After Completing

1. If a counter-balance was used, remove any weight from the weight post, then remove <span class="ref-cable">cable</span> <span class="ref-cable">C2</span> from hole **18** and return it to its storage hole.
2. Unclip and remove the <span class="ref-accessory">tricep push-down bar</span> <span class="ref-accessory">S8</span>.
3. Remove <span class="ref-cable">C3</span> from the smith carriage crossbar and attach to the storage for C3 pin.
