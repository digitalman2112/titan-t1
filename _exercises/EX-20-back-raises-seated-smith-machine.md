## Setup

1. Insert <span class="ref-accessory">exercise handle</span> <span class="ref-accessory">S1-L</span> into hole **12-R** and handle <span class="ref-accessory">S1-R</span> into hole **12-L** on the smith carriage. The grips should face inward.
2. Adjust the bench to position **1** (adjust to preference) and lock into place.
3. Adjust the backrest to the desired incline position using the pull pin located behind the seat. Position the backrest at a height where the arms are extended from the seated position, then lock into place.
4. If the weight of the smith carriage without added weight is too heavy, insert <span class="ref-cable">counter-balance cable</span> <span class="ref-cable">C2</span> into hole **18** and add the desired counter weight.
5. Add the desired weight, distributed evenly on both sides.

## Back Raises

1. Grasp the <span class="ref-accessory">exercise handles</span> firmly and unlock each side.
2. Lift the weight with the shoulders and arms in an upward direction.
3. Pause briefly and return to the starting position.
4. When finished, lock both <span class="ref-accessory">handles</span> securely onto the <span class="ref-hook">hooks</span>.

## After Completing

1. If a counter-balance was used, remove any weight from the weight post, then remove <span class="ref-cable">cable</span> <span class="ref-cable">C2</span> from hole **18** and return it to its storage hole.
2. Unlock the <span class="ref-accessory">exercise handles</span> and lower the smith carriage to rest on the springs.
3. Remove weights from the smith carriage posts.

## Precautions

- Always ensure **both** <span class="ref-accessory">handles</span> are securely locked before starting and when finishing. Never secure the smith carriage by locking only one side.
- After the bench is properly positioned, ensure that it is locked into place.
