## Setup

1. Using an attachment clip, attach <span class="ref-cable">cable</span> <span class="ref-cable">C8</span> to the attachment bar located on the inside of the counter-balance.
2. Attach <span class="ref-cable">extension cable</span> <span class="ref-cable">C5</span> to the other end of <span class="ref-cable">cable</span> <span class="ref-cable">C8</span>.
3. Attach <span class="ref-cable">cable</span> <span class="ref-cable">C10</span> to <span class="ref-cable">cable</span> <span class="ref-cable">C5</span>.
4. Attach the <span class="ref-accessory">curl bar</span> <span class="ref-accessory">S8</span> handle to <span class="ref-cable">cable</span> <span class="ref-cable">C10</span>.

## Standing Cable Curls and Upright Rows

1. Grab the <span class="ref-accessory">handle</span> <span class="ref-accessory">S8</span> firmly with both hands and begin the exercise.
2. For curls, curl the bar toward the upper chest in a slow, controlled manner. For upright rows, pull the bar upward toward the chin keeping the elbows high.
3. Pause briefly in the fully contracted position and return to the starting position.
4. Instructions can also be found on the video CD that was included with your machine.

## Precautions

- Be sure that all cables are properly and securely attached before use.
