<span class="ref-cable">C10</span> is an intermediate cable used to extend the cable run in Option F exercises:

- For **standing cable curls/upright rows**: attach to <span class="ref-cable">C5</span>, then attach handle <span class="ref-accessory">S8</span> to the end of <span class="ref-cable">C10</span>.
- For **abductor/adductor**: attach to <span class="ref-cable">C9</span>, then attach the ankle strap to the end of <span class="ref-cable">C10</span>.
